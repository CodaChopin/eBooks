# v-mask

> 限制輸入框格式元件，例如當使用者輸入電話號碼"0911123456"，可主動讓其輸入值轉換為"0911-123-456"

## Github

[vuejs-tips/vue-the-mask](https://github.com/probil/v-mask)

## 格式限制指定值

| 符號 | 符合值域 |
|:----:|:--------|
| # | 數字 |
| S | 字母 (a-z, A-Z) |
| X | 數字或字母 (0-9, a-z, A-Z) |
| A | 字母 (a-z, A-Z)，且轉為大寫 (toLocaleUpperCase()) |
| a | 字母 (a-z, A-Z)，且轉為小寫 (toLocaleLowerCase()) |
| ! | Enable escape |
